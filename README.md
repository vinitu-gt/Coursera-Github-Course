# Simple Interest Calculator

A lightweight bash script that calculates simple interest based on user input for principal, annual rate of interest, and time period in years.

## Formula
The formula used for calculating simple interest is:
   **SI = (P * R * T) / 100**

Where:
* **P** = Principal amount
* **R** = Annual rate of interest
* **T** = Time period in years

## Usage
1. Clone the repository to your local machine.
2. Ensure the script has execution permissions: `chmod +x simple-interest.sh`
3. Run the script: `./simple-interest.sh`
4. Enter the required values when prompted to see the calculated interest.
