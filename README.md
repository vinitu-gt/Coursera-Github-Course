# Simple Interest Calculator

A lightweight web application/script that calculates simple interest based on user input.

## Features
* Calculate total interest earned over a period.
* Calculate the final balance (Principal + Interest).
* User-friendly interface (or CLI).

## Formula Used
The calculator uses the standard simple interest formula:
I = (P × R × T) / 100

## How to Use
1. Clone the repository.
2. Open `index.html` (or run `main.py`).
3. Enter the Principal, Rate, and Time.
4. Click 'Calculate'.
